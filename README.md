### The Facility (Funding Layer)
Crowdfunding multimedia projects through smart contracts.

**How it works:**
- Creators sell % of project to "Facilitators" (investors)
- Milestone-based funding with escrow protection
- Secondary market for trading project ownership

### DIGI (Distribution Layer)
Decentralized video/audio encoding and delivery.

**How it works:**
- Powered by Livepeer + Audius (no data centers!)
- Micro-transactions (pay per view, NO subscriptions)
- 10% fee (5% encoders, 5% DIGI)
- Token launch ready
